**Awesome Selfhosted** is a beautifully designed, interactive dashboard for discovering the best open-source and self-hostable software. Take control of your data, enhance your privacy, and build your own digital ecosystem.

[**🌐 Live Demo**](https://ishandutta2007.github.io/Awesome-Selfhosted/)

---

## ✨ Features

- **🔍 Smart Search:** Instant filtering across 160+ software entries.
- **📂 Categorized Discovery:** 29+ categories ranging from Analytics to VPNs.
- **🌓 Dark Mode Support:** Seamlessly switch between light and dark themes.
- **📱 Fully Responsive:** Optimized for desktop, tablet, and mobile viewing.
- **⭐ GitHub Integration:** Real-time star counts and direct links to source code.
- **🏷️ Rich Metadata:** At-a-glance information on licenses, languages, and deployment platforms.
- **🚀 Ultra Fast:** Built with vanilla HTML/JS/CSS for maximum performance.

---

## 🛠️ Tech Stack

- **HTML5:** Semantic structure for better accessibility and SEO.
- **CSS3:** Modern layouts with Flexbox/Grid and smooth transitions.
- **JavaScript (ES6+):** Dynamic rendering and efficient state management.
- **Font Awesome:** Beautiful iconography for a professional look.

## 📂 Software Categories

Our curated list covers every aspect of self-hosting:

- **Productivity:** Note-taking, Office suites, Project Management.
- **Media:** Streaming, Photo galleries, RSS readers.
- **Infrastructure:** Networking, VPNs, Monitoring, Security.
- **Development:** Code editors, Database management, Automation.
- **And much more!**

## 🤝 Contributing

We love contributions! If you know of any awesome self-hosted software that is missing:

1.  Fork the repository.
2.  Add the software entry to `data.js` following the existing format.
3.  Ensure the icon and metadata are correct.
4.  Submit a Pull Request.
